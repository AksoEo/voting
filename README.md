# voting systems
This library implements the following voting systems according to [TEJO regulation](https://github.com/tejoesperanto/dokumentoj/blob/master/dok/Reglamentoj/Reglamento%20pri%20voÄ‰donadoj.md):

- ranked pairs
- single transferable vote

It also implements:

- yes/no
- yes/no/blank
- UEA threshold majority

Ballots are not validated.

## Usage
Build with `npm install; npm run build`. Javascript output will be in `dist/`.
